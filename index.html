Reinforcement learning with Matlab r2019b

Teaching a snake AI to eat lots of food.

This 2D Snake project is designed to introduce reinforcement learning concepts using an Actor-Critic algorithm. In this game, the snake eats as much food as possible without hitting the boundaries or itself. The neural network has sixteen input neurons, and four output neurons. While training the algorithm, a progress graph will display the effectiveness of the training. A simulation of the snake also diplays in a figure to help visualize what actions the network takes.
